# STEP File Browser

A simple file browser for navigating the relationships between ISO 10303 (STEP)
files.

## Dependencies

* python
* libpython2.6
* python-matplotlib
* python-networkx
* python-numpy
* python-scipy
* python-qt4
* python-qt4-dev
* libqtgui4
* libqtcore4
* qt4-dev-tools
